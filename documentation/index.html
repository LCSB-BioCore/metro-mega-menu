Installation

First you need to include the css and img folder in your website directory.

Then you need to load the css file, to do that add this code in the <head> of your html document.

<link href="css/metro.css" rel="stylesheet" media="screen" type="text/css">

To finish you need to add the metro menu HTML markup in the <body> of your html document.

<ul class="metro">
    ...
</ul>

    <li class="small-tile-image rotate180">
        <a class="tile" href="#">
            <span>Animation</span>
            <img src="../menu/img/icons/metro/icon.png" alt="">
        </a>
    </li>
</ul>
Class Description
.rotate360 Rotate the image 360deg.
.rotate180 Rotate the image 180deg.
.scale10 Scale the image 10%.
.scale-10 Scale the image negative 10%.
.flip Flip the image.

.drop-2-col Box size for two columns.

.drop-4-col Box size for four columns.
.col-1 One column size, to use inside of the dropdown box.

Box Positions

        </div>

Class Description
.left Align the dropdown box from left to right. No class is required.
.center Align the dropdown box on the center.
.right Align the dropdown box from right to left.
.right-top Align the dropdown box on the right of the tile, from top to bottom.
.right-bottom Align the dropdown box on the right of the tile, from bottom to top.
.left-top Align the dropdown box on the left of the tile, from top to bottom.
.left-bottom Align the dropdown box on the left of the tile, from bottom to top.

Custom Colors

You can create unlimited colors for the menu tiles. To do that you need to change the class name and the color value from the snippet bellow.

ul.metro li.yourColorClass a.tile { background-color: #ffffff; }

Color Generator
